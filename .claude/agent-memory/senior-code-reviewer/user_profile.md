## 기술 수준

- Next.js 16 App Router, React 19, TypeScript strict 기반 프로젝트를 직접 설계하고 구축한 수준
- 보안 헤더, SSR-safe hydration(useSyncExternalStore), Tailwind v4 @theme 블록 등 고급 기법을 자연스럽게 적용
- Zustand, Zod, react-hook-form, shadcn/ui 등 현대 프론트엔드 표준 스택에 능숙

## 프로젝트 맥락

- 이 저장소는 한국어 친화 Next.js 스타터킷 — 랜딩·문서·예제 위주, API/DB 없음
- 스타터킷 특성상 일부 데모 코드(setTimeout, 더미 URL)는 의도적 트레이드오프

## 커뮤니케이션 스타일

- 커밋 메시지·주석·문서: 한국어
- 변수/함수명: 영어 camelCase/PascalCase
- 이모지 커밋 컨벤션 사용 (✨ Feat, 🐛 Fix, ⚡️ Add 등)
